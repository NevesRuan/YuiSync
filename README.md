# YuiSync
Automated short-form video syndication system built with Spring Boot. It acts as a centralized bridge to fetch content and autonomously distribute it across multiple social media platforms.

## Prerequisite
⚠️ yt-dlp should be installed and available in your $PATH.

[How to properly install yt-dlp executable](https://github.com/yt-dlp/yt-dlp#installation)

Otherwise you will get this error :

yt-dlp is not available in PATH. Please install yt-dlp and ensure it's added to your system's PATH.

### Example for .env
``` 
DB_USERNAME=your_user_here
DB_PASSWORD=your_password_here
```
